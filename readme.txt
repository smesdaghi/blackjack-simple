
Notes
=====
The project is built using gradle.
To run the game execute ./run.sh
To run the tests execute ./test.sh

alternatively, you can run the following:
java -jar ./build/libs/blackjack-1.0.jar

Dependancies
============
dependencies/hamcrest-core-1.3.jar
dependencies/junit-4.11.jar


Tested On
=========
OSX 10.10, jdk 1.7

Ubuntu 12.04, jdk 1.7
  - suit and hidden card symbol draw as '?' when the font being used doesn't support the symbols

Windows 7, jdk 1.7
  - suit and hidden card symbol draw as '?' when the font being used doesn't support the symbols
